# Zomato_Data_Analysis
**Exploratory Data Analysis of Zomato data**
- This project focuses on explatory data analysis (EDA) of Zomato restaurant data to uncover meaningful insights about restaurants, cuisines, pricing, ratings, and customer preferences.
The goal is to analyze trends and patterns that can help businesses and customers make better decisions.
<img width="1280" height="720" alt="image" src="https://github.com/user-attachments/assets/f504489b-e9e4-4f38-992e-382b8e3ee6e9" />


**Key Objectives**
- Understand restaurant distribution across locations
- Analyze customer ratings and reviews
- Identify popular cuisines
- Study price range vs ratings relationship
- Discover factors influencing restaurant success

**Dataset Source:** Zomato Dataset
Contains information such as:
- Restaurant Name, Location, Cuisines, Cost for two, Ratings, Votes etc.

**Technologies Used:**
- Python
- Pandas → Data manipulation
- NumPy → Numerical operations
- Matplotlib & Seaborn → Data visualization
- Jupyter Notebook → Analysis environment

<img width="1280" height="720" alt="Presentation1" src="https://github.com/user-attachments/assets/24598b0e-49f2-49d0-b94b-5a98db30cf78" />


**Analysis Performed:**
- Data Cleaning
- Handling Missing Values
- Feature Understanding
- Univariate Analysis
- Bivariate Analysis
- Correlation Analysis
- Visualization of Insights

Key Insights
- Highly rated restaurants are not always expensive
- Mid-range pricing attracts more customers
- Votes and ratings are strongly correlated
